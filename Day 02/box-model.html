CSS Box Model & Display Properties

1) Box Model

Content Area
Padding: 15px
Border: 5px
Margin: 20px

Explanation: Content is inside → Padding adds space inside border → Border surrounds padding → Margin adds space outside element.

2) Display: Inline, Block, Inline-Block

Inline Inline-Block Block

Inline: Does not break line, cannot set width/height.
Block: Takes full width, starts new line.
Inline-block: Inline flow + can set width/height.

5) Quick Cheat Sheet

ConceptNotes
ContentActual data/text inside element.
PaddingSpace inside border. Affects background.
BorderLine around padding. Example: border:2px solid.
MarginSpace outside border. Separates elements.
Displayinline / block / inline-block / flex / grid.
Flex1D layouts. Use display:flex.
Grid2D layouts. Use display:grid.
